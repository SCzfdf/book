堆 : 存放所有new出来的对象。
栈 : 存放  基本类型  的变量数据和对象的引用,
	但对象本身不存放在栈中,而是存放在堆(new 出来的对象)或者常量池中(字符串常量对象存放在常量池中。)
常量池(域) : 存放字符串常量 和 基本类型常量(public static final)。


还有其他三个存放数据的区域分别是 寄存器, 静态域, 非RAM储存
寄存器 : 最快的存储区, 由编译器分配, 无法操作,无法在编程中感知
静态域(池) : 存放静态成员(static修饰)
非RAM存储 : 硬盘 存放持久化的对象

速度 : 寄存器 > 栈 > 堆

内存机制 : 
  栈 -> Java栈内存以帧的形式存放, 每调用一个方法就将对应该方法的方法帧压入 Java 栈,成为当前方法帧
	方法帧包含了: 方法调用的参数,局部变量,中间结果等
	当调用结束时,  马上  弹出该帧
